**🇷🇺 «Большая битва отрядов»**

Этот репозиторий посвящён разработке консольной игры на Python. Здесь я шаг за шагом превращаю первоначальную идею в полноценный проект.

Игра создаётся постепенно — от самой простой рабочей версии к более сложной.

---

## 📖 О проекте

В игре игрок и компьютер распределяют по **40 воинов** между **5 отрядами**.

После этого каждый отряд сражается с соответствующим отрядом противника.

За каждую победу начисляется одно очко.

Побеждает тот, кто выиграет больше сражений.

The goal of this repository is not only to create the game itself, but also to document the entire development process.

Along the way, I record not only the source code, but also design decisions, experiments, mistakes, ideas, and the small discoveries that appear during development.

---

## 📖 About the Project

In this game, both the player and the computer distribute **40 warriors** among **5 squads**.

Each squad then battles the corresponding enemy squad.

Every winning battle awards one point.

The side with the most points wins the game.
